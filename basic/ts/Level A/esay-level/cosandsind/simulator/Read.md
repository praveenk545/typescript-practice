# Physics & Math Based Programming Projects

This repository contains physics-inspired simulations built using
JavaScript, HTML Canvas, and mathematical modeling.

## Projects

1. Gravitational Field Visualizer
2. Stable Orbit Simulator
3. Wave Interference Simulator
4. Gravitational Lensing Simulator
5. N-Body Chaos Simulator

## Purpose

- Learn trigonometry (sin, cos)
- Understand vectors and forces
- Simulate real-world physics
- Build intuition for gravity, waves, and chaos

## Tech Stack

- HTML5 Canvas
- JavaScript
- CSS
